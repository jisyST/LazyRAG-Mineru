# LazyMiner
LazyMiner 是一个结合 MinerU 和 LazyLLM 的 PDF 解析与 RAG 集成工具。

## 📂 目录结构
- **config/** – 配置文件  
- **data/** – 示例 PDF 文件  
- **models/** – 模型文件  
- **scripts/** – 安装、下载脚本  
- **src/** – 核心源代码  
- **tests/** – 测试用例  

## 🚀 快速启动
```bash
# 安装依赖
pip install -r requirements.txt

# 运行项目
python src/main.py
```
